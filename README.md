## Sistema de Calificación Estudiantil

## Descripción
Este proyecto consiste en una aplicación de escritorio que permite ingresar cuatro notas de un estudiante, calcular el promedio y determinar si el estudiante debe ir a completivo o extraordinario según su calificación.

## Tecnologías Utilizadas
- Lenguaje: C#
- Framework: Windows Forms (.NET)
- Entorno / IDE: Visual Studio
- Base de datos: No aplica

## Uso o Ejecución
1. Abrir el proyecto en Visual Studio
2. Ejecutar el programa
3. Ingresar las cuatro notas
4. Calcular el promedio
5. Visualizar el resultado final del estudiante
